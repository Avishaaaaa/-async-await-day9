User Directory App

Overview

User Directory App is a simple web application built using HTML, CSS, and JavaScript. It fetches user data from an external API and displays it in a user-friendly format. Users can search for specific users by name or username in real time.

Features
Fetch user data from an API
Display user information dynamically
Search users by name or username
Loading message while data is being fetched
Error handling using try/catch
Responsive card-based layout

API Used

How It Works
When the page loads, user data is fetched from the API.
A loading message is displayed while the request is in progress.
User information is shown in cards after the data is received.
Users can search by name or username.
Matching results are displayed instantly.
If an error occurs, an error message is shown.
